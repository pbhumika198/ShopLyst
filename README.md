#ShopLyst- E-Commerce React App

A modern and responsive **E-commerce Web Application** built using React.js.
This project demonstrates real-world shopping features like product browsing, cart management, and dynamic U
## Features

 Browse products from API
 Search products in real-time
 Add to Cart functionality
 Increase / Decrease quantity
 Remove items from cart
 Product Details Page
 Fast and responsive UI

---

## Tech Stack

* React.js
* CSS3
* REST API (DummyJSON)
* React Router DOM
* Context API (State Management)

## Installation & Setup

1. Clone the repository
git clone https://github.com/pbhumika198/ShopLyst.git
2. Navigate to project folder
cd ShopLyst
3. Install dependencies
npm install
4. Run the app

npm start
